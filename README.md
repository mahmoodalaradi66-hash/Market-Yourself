## How to View This Portfolio

The portfolio cannot be viewed directly on GitHub as it includes local video files and images that require the files to be run together from your computer.

Please follow the steps below:

---

### Step 1 — Download the ZIP file

Click the green **Code** button at the top of this repository page, then click **Download ZIP**.

### Step 2 — Extract the ZIP file

Once downloaded, right-click the ZIP file and select:
- **Extract All** on Windows
- **Open With → Archive Utility** on Mac

### Step 3 — Open the portfolio

Open the extracted folder and double-click **index.html**.

The file will open in your default web browser and the full portfolio will load, including all pages, images, and videos.

---

## Notes

- All files must remain inside the same folder after extraction — do not move individual files separately or the links between pages will break.
- The portfolio is best viewed in **Google Chrome** or **Microsoft Edge**.
- No internet connection is required once the files are extracted.
